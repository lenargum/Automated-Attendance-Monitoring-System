### How to start server
#### Execute commands:

*Unix Bash (Linux, Mac, etc.)*:
```
$ export FLASK_APP=run.py
$ flask db upgrade
$ flask run -h 0.0.0.0
```
*Windows CMD*:
```
> set FLASK_APP=run.py
> flask db upgrade
> flask run -h 0.0.0.0
```
*Windows PowerShell*:
```
> $env:FLASK_APP = "run.py"
> flask db upgrade
> flask run -h 0.0.0.0
```
