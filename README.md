### How to start server:
#### Configure `FLASK_APP` to make it runnable from `run.py` inside project directory and execute `flask run`:

*Unix Bash (Linux, Mac, etc.)*:
```
$ export FLASK_APP=run.py
$ flask run
```
*Windows CMD*:
```
> set FLASK_APP=run.py
> flask run
```
*Windows PowerShell*:
```
> $env:FLASK_APP = "run.py"
> flask run
```
